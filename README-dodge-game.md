# Block Survivor (Python)

Updated to your request:
- Relics removed
- Added **20 unique powerups**
- Right-side **mouse power panel**
- **Scroll with mouse wheel**
- **Click powerups with mouse** to buy/use
- Bought toggle/passive powers activate instantly

## Controls
- Move: `WASD` / Arrow keys
- Dash + spear hit: `Space`
- Restart: `R`
- Mouse wheel: scroll power list
- Mouse left click: buy/use selected power in panel

## Run
```bash
python dodge_game.py
```
